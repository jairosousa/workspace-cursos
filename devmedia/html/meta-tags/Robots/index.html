Robots

Para definir se uma página deve ou não ser exibida em um buscador, precisamos utilizar a tag <meta name="robots"> dentro do cabeçalho HTML e é isso que vamos aprender nessa aula

noindex

A meta tag noindex é utilizada quando queremos que uma página não apareça nos resultados de busca.

Fazendo o uso correto do noindex, os buscadores vão saber quais páginas devem ser exibidas nos seus resultados de busca.

O uso da meta tag noindex também é importante em páginas que possuem dados pessoais

no index é usada assim

<name="robots" content="noindex">

Nofollow

Além do noindex, existe também um outro tipo de uso para a tag robots, o nofollow.

Todo buscador acessa o conteúdo de páginas em busca de informações, como título, descrição e links.

Sem a mesta tag nofollow, o buscador vai navegar por esse links e associá-los a nossa página. Nesse caso é o que queremos, pois sãp links internos.

A meta tag nofollow diz ao buscador para não associar os links da página a nós, isso é útil quando a página possui muitos links externos, ou quando os links apontam para páginas com dados pessoais, extratos, numeros de telefone, etc...

Noindex, Nofollow

Legal, já sabemos utilizar o noindex e o nofollow e qual a utilidade de cada um. Mas e se a gente precisar usar os dois ao mesmo tempo?

<name="robots" content="noindex, nofollow">
